# Darwin Cybersecurity Toolkit

A Python cybersecurity toolkit that combines multiple security tools into one application. This project demonstrates basic cybersecurity concepts, networking knowledge, log analysis, and threat detection using Python.

## Features

- Password Strength Checker
- IP Address Checker
- URL Checker
- Log Analyzer
- Security Alert Checker

## Code Features

### Password Checker
- Detects weak passwords
- Encourages stronger password practices

### IP Checker
- Detects Localhost
- Identifies Private Network IPs
- Identifies Public IPs

### URL Checker
- Detects suspicious shortened URLs
- Identifies potential phishing links

### Log Analyzer
- Detects failed login attempts
- Introduces basic SOC analyst concepts

### Security Alert Checker
- Detects malware keywords
- Detects virus keywords
- Detects ransomware keywords
